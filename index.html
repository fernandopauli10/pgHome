HTML é a linguagem padrão utilizada para criar e estruturar páginas web

Ele organiza o conteúdo em elementos que formam a base de um site

como textos, imagens, links e botoes. Cada parte do documento é

representada por tags que indicam ao navegador como renderizar o conteúdo
